华软学院启动“学雷锋服务月” 植树节传承“百年树人”精神

本网讯 3月12日，我院一年一度的“学雷锋服务月”暨“百年树人”植树节活动启动仪式在喷泉广场举行，学院院长何大进、党委书记刘友坤、党委副书记张高峰、副院长康忠理以及各系部负责人、师生代表出席了启动仪式。

“只要我们心存雷锋精神，投身社会实践，人人皆是雷锋”，启动仪式上，学院党委张高峰副书记提到，“雷锋精神是我们共同拥有的精神财富，随着时代的发展，雷锋精神愈加珍贵。”张高峰副书记强调，学雷锋不该流于形式，而要注重实效，从而促进学雷锋志愿服务活动常态化开展。

在启动仪式现场，17级软件工程系志愿者代表姚宇豪同学分享了他的志愿者经历，“帮助别人，其实也在潜移默化地改变自己”，他表示志愿活动让他改变了生活态度，学会了乐观地生活与积极地工作，他还号召同学们积极参与到“学雷锋服务月”活动中来。在全体嘉宾起立宣誓和合唱《学习雷锋好榜样》后，“学雷锋服务月”暨“百年树人”活动正式宣布启动。

“百年树人”植树节传承扎根大地的精神 “百年树人”植树节系列活动是华软自建校以来就有的一项传统活动，17年来伴随着华软人不断成长壮大，植树节不仅传承华软“扎根大地”生生不息精神。 记者看到，学院领导、老师、同学们纷纷参与到以部门为单位留名认种树苗的队伍中来。活动现场还举办了“填诗词赢盆栽”“除草护树美化校园”“室内盆栽养护知识咨询”“绿色与健康同行”和“华软树叶”书签制作发放等活动，现场师生踊跃参与。此次活动蕴含着华软人寄情于树木，融情于华软的情怀，同时也令全校师生对“十年树木，百年树人”的传统有了切身的感受。
